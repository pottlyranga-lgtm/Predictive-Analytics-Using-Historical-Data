## 🎯 Project Overview
A Predictive Analytics project that uses Machine Learning models to forecast future revenue trends, evaluate model accuracy, and generate actionable business insights from historical sales data.

## 🚀 Features
| Feature | Description |
|---------|-------------|
| 🧹 Data Cleaning | Handle missing values, duplicates, data types |
| ⚙️ Feature Engineering | Month, Quarter, Season encoding |
| 📊 EDA | Revenue trends, correlations, category analysis |
| 🤖 Linear Regression | Baseline predictive model |
| 🌲 Random Forest | Advanced ensemble model |
| 📏 Model Evaluation | R², MAE, RMSE comparison table |
| 📈 Actual vs Predicted | Visual model performance comparison |
| 🔮 Time Series Forecast | Next 3 months prediction with confidence interval |
| 💡 Business Insights | Auto-generated recommendations |

## 🤖 Models Used
- **Linear Regression** — Baseline revenue prediction
- **Random Forest Regressor** — 100 estimators, high accuracy
- **Time Series Forecasting** — 3-month future revenue forecast

## 🛠️ Technologies Used
- Python, Pandas, NumPy
- Scikit-learn
- Plotly
- Jupyter Notebook
